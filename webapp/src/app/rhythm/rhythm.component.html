Practice Rhythms

Overview

Rhythm is the driving foundation of modern music. In a tight groove even wrong notes sound right, which means that good timing is more important than good note choices. This deck contains the most common rhythmic patterns.

Cards are sorted by difficulty, where difficulty is the complexity and tempo of the rhythm. So, you probably don't want to configure Anki to show cards in random order (unless you really want to).

Description

Fields

The Type field can have the following values:

  • Basic

    There are 8 ways to decompose a quarter note into 8th, 16th and 8th-triplets without introducing rests. These are shown here:

    These are the building blocks that make up many rhythms. Exercises with Type=Basic repeat one of these blocks, so you get familiar with them and recognize them in more complex, composite rhythms.

  • Syncopated

    Syncopated rhythms do not align with the downbeat of the measure. Again there are 8 basic syncopated building blocks which are derived from the above by replacing the first beat with a rest:

    Exercises with Type=Syncopated repeat one of these blocks.

  • Tied

    Tied rhythms sound very similar to syncopated rhythms, but notation is quite different.

    Exercises with Type=Tied repeat one of these blocks.

  • Standard

    These cards contain some standard rhythms / patterns that are very common, like Claves, Bossas, Charleston and others.

  • Grouping

    Groupings are another way to create intersting rhythmic patterns. Take a pattern of length three 16th notes and repeat that pattern for a whole measure. Because 3 does not fit into 16 there is some extra beat when the measure repeats. A grouping of 3 in position 1 looks like this:

  • Random

    This picks randomly some of the Basic and some of the Syncopated building blocks and combines them into new rhythms. Also some quarters are randomly tied.
